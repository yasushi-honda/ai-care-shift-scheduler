# Product Context - AIシフト自動作成

## プロダクト概要

**名称**: AIシフト自動作成（AI Care Shift Scheduler）
**対象**: 介護・福祉事業所向けシフト管理システム
**目的**: Gemini AIを活用した効率的なシフト作成とスタッフ管理

## ビジネス目標

### 主要目標
1. **業務効率化**: 手動シフト作成の時間を80%削減
2. **最適化**: AIによる公平で効率的なシフト配置
3. **コンプライアンス**: 労働基準法と介護保険制度の遵守
4. **使いやすさ**: ITリテラシーが低いユーザーでも直感的に操作可能

### 成功指標（KPI）
- シフト作成時間: 従来の4時間 → 30分以内
- スタッフ満足度: 休暇希望の90%以上を反映
- 人員配置適正率: 法定基準を100%満たす
- ユーザー継続率: 6ヶ月後80%以上

## ターゲットユーザー

### プライマリユーザー
- **管理者・施設長**: シフト作成、承認、最終調整
- **サービス提供責任者**: 日々のシフト確認と調整
- **事務職員**: スタッフ情報管理、データ入力

### 特徴
- 年齢層: 30代〜60代
- ITリテラシー: 初級〜中級
- 業務環境: PC/タブレット使用、忙しい現場

## コアバリュー

### 1. AI支援によるスマートな自動化
- Gemini 2.5 Pro（asia-northeast1）を使用
- スタッフのスキル、資格、希望を考慮
- 法令遵守を保証する制約条件

### 2. シンプルで直感的なUI
- 左パネル: スタッフ情報・要件設定
- 右パネル: カレンダービュー
- ワンクリックでシフト生成

### 3. 柔軟な調整機能
- 手動での微調整が可能
- 休暇申請の即時反映
- バージョン管理による履歴追跡

### 4. データ活用
- CSV形式でのエクスポート
- 過去のシフトパターン分析
- レポート機能（将来実装）

### 技術的制約
- ブラウザのみ: PC/タブレット推奨、スマホは非推奨
- オフライン非対応: インターネット接続必須

## ビジネスルール

### 介護事業所のシフト要件
1. **時間帯区分**
   - 早番（7:00-16:00）
   - 日勤（9:00-18:00）
   - 遅番（11:00-20:00）
   - 夜勤（17:00-9:00）

2. **人員配置基準**
   - 利用者10名あたり介護職員1名以上
   - 看護職員は日中必須
   - 夜勤は2名体制推奨

3. **資格要件**
   - 介護福祉士
   - 看護師・准看護師
   - 普通自動車免許（送迎業務）

4. **労働基準法遵守**
   - 週40時間以内の労働
   - 月8回以上の休日
   - 連続勤務は最大6日まで
   - 夜勤後の休息時間確保

## 用語集

| 用語 | 説明 |

| 休暇申請 | スタッフが勤務できない日の登録 |
| 夜勤専従 | 夜勤のみを担当するスタッフ |
| タグループ | シフト表示のグループ分け機能 |

## 競合分析

### 既存のシフト管理ツール
- **強み**: 実績豊富、多機能
- **弱み**: AI活用が弱い、介護特化が少ない

### 当プロダクトの差別化要因
1. **最新AI技術**: Gemini 2.5 Pro（asia-northeast1）
2. **介護業界特化**: 法令遵守を自動保証
3. **低コスト**: Firebase/GCP活用でインフラコスト削減
4. **4段階評価システム**: 制約レベル別の品質評価

## リスクと対策

### 技術的リスク
- **AIの精度**: デモモードで代替可能
- **スケーラビリティ**: Cloud Functionsの自動スケーリング
- **データ保護**: Firestore Security Rulesで保護（認証後）

### ビジネスリスク
- **法令変更**: 柔軟な要件設定で対応可能
- **ユーザー獲得**: 無料トライアルとデモ環境提供

## まとめ

AIシフト自動作成は、介護・福祉事業所の管理者が抱えるシフト作成の負担を、最新のAI技術で劇的に軽減するプロダクトです。シンプルで使いやすいUIと、業界特有の制約を理解したAIエンジンにより、誰でも簡単に最適なシフトを作成できます。
