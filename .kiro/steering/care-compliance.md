# 介護報酬算定ガイドライン

**作成日**: 2025-11-20
**対象**: Phase 25 介護報酬対応 - 予実管理機能実装
**目的**: 介護報酬算定の基礎知識、人員配置基準、常勤換算計算、労基法遵守項目を記録

---

## 1. 介護保険制度の基礎

### 1.1 介護保険とは

**概要**: 高齢者の介護を社会全体で支え合う仕組み

**対象者**:
- 65歳以上の方（第1号被保険者）
- 40歳以上65歳未満の医療保険加入者（第2号被保険者、特定疾病の場合のみ）

**財源**:
- 公費: 50%（国25%、都道府県12.5%、市町村12.5%）
- 保険料: 50%（第1号被保険者22%、第2号被保険者28%）

### 1.2 介護報酬とは

**定義**: 介護サービス事業者が利用者にサービスを提供した際に、介護保険から支払われる対価

**報酬の決定**:
- 厚生労働大臣が定める
- 3年ごとに改定（直近: 令和6年度・2024年度改定）

**算定要件**:
- 人員基準: 必要な専門職の配置
- 設備基準: 適切な施設・設備
- 運営基準: サービスの質の確保

## 2. 人員配置基準

### 2.1 人員配置基準とは

**定義**: 介護サービスを提供するために必要な職員の配置人数

**目的**:
- サービスの質の確保
- 利用者の安全確保
- 職員の労働環境改善

### 2.2 主なサービスの人員配置基準

#### 訪問介護

| 職種 | 配置基準 |
|------|---------|
| 管理者 | 常勤専従1名以上（他の職務と兼務可能） |
| サービス提供責任者 | 常勤の訪問介護員等のうちから1名以上<br/>利用者40人ごとに1名追加 |
| 訪問介護員 | 常勤換算2.5人以上 |

#### 訪問看護

| 職種 | 配置基準 |
|------|---------|
| 管理者 | 常勤専従1名以上 |
| 看護職員 | 常勤換算2.5人以上 |
| 理学療法士等 | 常勤換算1人以上（訪問看護ステーションの場合） |

#### 通所介護（デイサービス）

| 職種 | 配置基準 |
|------|---------|
| 管理者 | 常勤専従1名以上 |
| 生活相談員 | 提供時間帯を通じて専ら従事する者が1名以上 |
| 看護職員 | 単位ごとに専ら従事する看護職員または介護職員が1名以上 |
| 介護職員 | 利用者数に応じて配置<br/>利用者15人まで: 1名以上<br/>利用者15人超: 利用者数を5で除した数+1人以上 |
| 機能訓練指導員 | 1名以上 |

#### 特別養護老人ホーム（介護老人福祉施設）

| 職種 | 配置基準 |
|------|---------|
| 医師 | 1名以上 |
| 生活相談員 | 入所者100人に対し1名以上 |
| 介護職員・看護職員 | 利用者3人に対して介護職員・看護職員1名以上（3:1）<br/>うち看護職員は利用者30人に対し1名以上 |
| 栄養士 | 入所定員40人以上の場合1名以上 |
| 機能訓練指導員 | 1名以上 |
| 介護支援専門員 | 1名以上（入所者100人に対し1名追加） |

### 2.3 配置基準の計算例

#### 例1: 特別養護老人ホーム（入所者90人）

**3:1配置基準**:
- 必要人員: 90人 ÷ 3 = **30人**
- 介護職員・看護職員の合計が常勤換算30人以上必要

**実際の配置（例）**:
- 介護職員: 常勤換算25人
- 看護職員: 常勤換算5人
- **合計: 30人** → ✅ 基準達成

#### 例2: 通所介護（利用者25人）

**配置基準**:
- 介護職員: 利用者15人超の場合、利用者数 ÷ 5 + 1人
- 計算: 25 ÷ 5 + 1 = **6人**

**実際の配置（例）**:
- 介護職員: 7人
- **充足率**: 7 ÷ 6 × 100 = 116.7% → ✅ 基準達成

## 3. 常勤換算とは

### 3.1 常勤換算の定義

**定義**: 非常勤職員を含めた全職員の勤務時間を、常勤職員の勤務時間で割って算出した数値

**目的**:
- 非常勤職員を含めた実際の人員配置を正確に把握
- 人員配置基準の充足状況を確認

### 3.2 常勤換算の計算方法

**基本式**:
```
常勤換算値 = 職員の勤務時間 ÷ 当該事業所の常勤職員が勤務すべき時間
```

**週所定労働時間**:
- 一般的には週40時間（労働基準法の原則）
- 事業所ごとに就業規則で定められる

### 3.3 常勤換算の計算例

#### 例1: 訪問介護事業所（週所定労働時間40時間の場合）

**職員の勤務状況（1ヶ月 = 4.33週）**:

| 職員名 | 雇用形態 | 週勤務時間 | 月勤務時間 | 常勤換算値 |
|--------|---------|-----------|-----------|-----------|
| A | 常勤 | 40時間 | 173.2時間 | 1.00 |
| B | 常勤 | 40時間 | 173.2時間 | 1.00 |
| C | 非常勤 | 20時間 | 86.6時間 | 0.50 |
| D | 非常勤 | 16時間 | 69.3時間 | 0.40 |
| **合計** | - | - | - | **2.90** |

**計算式**:
- A, B: 40時間 ÷ 40時間 = 1.00
- C: 20時間 ÷ 40時間 = 0.50
- D: 16時間 ÷ 40時間 = 0.40

**基準達成状況**:
- 訪問介護の基準: 常勤換算2.5人以上
- 実際: 2.90人
- **充足率**: 2.90 ÷ 2.5 × 100 = 116% → ✅ 基準達成

#### 例2: 月間実績勤務時間から常勤換算を計算

**前提**:
- 週所定労働時間: 40時間
- 1ヶ月 = 4.33週（平均）
- 月所定労働時間: 40 × 4.33 = 173.2時間

**職員の月間実績勤務時間**:

| 職員名 | 月間実績勤務時間 | 常勤換算値 |
|--------|---------------|-----------|
| 田中太郎 | 160時間 | 160 ÷ 173.2 = **0.92** |
| 佐藤花子 | 80時間 | 80 ÷ 173.2 = **0.46** |
| 鈴木一郎 | 173時間 | 173 ÷ 173.2 = **1.00** |
| 高橋美咲 | 120時間 | 120 ÷ 173.2 = **0.69** |
| **合計** | **533時間** | **3.07** |

## 4. 労働基準法の遵守項目

### 4.1 休憩時間

**労働基準法第34条**: 使用者は、労働時間が6時間を超える場合は少なくとも45分、8時間を超える場合は少なくとも1時間の休憩時間を与えなければならない。

**基準**:
- 勤務時間6時間以下: 休憩不要
- 勤務時間6時間超~8時間以下: 少なくとも45分
- 勤務時間8時間超: 少なくとも60分（1時間）

**例**:
- 8:00 ~ 16:00勤務（8時間）: 45分以上の休憩必要
- 8:00 ~ 17:00勤務（9時間）: 60分以上の休憩必要

**チェックロジック**（Phase 25.4で実装）:
```typescript
if (workHours > 8 && breakMinutes < 60) {
  // 違反: 8時間超の勤務には60分以上の休憩が必要
}
if (workHours > 6 && breakMinutes < 45) {
  // 違反: 6時間超の勤務には45分以上の休憩が必要
}
```

### 4.2 連続勤務制限

**介護労働安全衛生ガイドライン**: 連続勤務は7日以内を原則とする（一部事業所では10日以内も認められる）

**目的**:
- 職員の健康維持
- サービスの質の確保
- 労働安全衛生の確保

**例**:
- 連続勤務7日 → ⚠️ 翌日は休日とすべき
- 連続勤務10日 → ❌ ガイドライン違反

**チェックロジック**（既存機能を活用）:
```typescript
let consecutiveDays = 0;
for (const shift of staffSchedule.monthlyShifts) {
  if (shift.plannedShiftType !== '休') {
    consecutiveDays++;
    if (consecutiveDays > maxConsecutiveDays) {
      // 違反: 連続勤務制限超過
    }
  } else {
    consecutiveDays = 0;  // リセット
  }
}
```

### 4.3 勤務間インターバル

**働き方改革関連法**: 勤務終了後、次の勤務開始まで一定の休息時間（インターバル）を設ける

**推奨インターバル**: 最低8時間（労働時間等設定改善法の指針）

**目的**:
- 過重労働の防止
- 職員の健康維持
- ワークライフバランスの確保

**例**:
- 前日22:00勤務終了 → 翌日8:00勤務開始 → インターバル10時間 → ✅ OK
- 前日23:00勤務終了 → 翌日6:00勤務開始 → インターバル7時間 → ❌ 推奨未達成

**特に注意すべきケース**:
- 夜勤 → 早番の連続勤務
- 遅番 → 早番の連続勤務

**チェックロジック**（既存機能を活用）:
```typescript
const previousDayEndTime = getPreviousShift(staffSchedule, date)?.actualEndTime;
const currentDayStartTime = getCurrentShift(staffSchedule, date)?.actualStartTime;

if (previousDayEndTime && currentDayStartTime) {
  const intervalHours = calculateInterval(previousDayEndTime, currentDayStartTime);
  if (intervalHours < 8) {
    // 違反: 勤務間インターバル不足
  }
}
```

## 5. 標準様式第1号について

### 5.1 標準様式第1号とは

**正式名称**: 従業者の勤務の体制及び勤務形態一覧表

**目的**:
- 指定申請・変更届の際に提出
- 人員配置基準の充足状況を確認
- 全国統一の様式で行政手続きを効率化

**提出先**:
- 高知県外・広域: 高知県庁 長寿社会課
- 高知市内: 高知市役所 介護保険課

**提出方法**:
- 電子申請（GビズID利用）が原則
- 紙での提出も可能

### 5.2 標準様式の項目

**必須項目**:
1. 施設名・事業所名
2. 対象月
3. 職員氏名
4. 職種・役職
5. 資格
6. 常勤・非常勤の別
7. 勤務シフト（日付ごと）
8. 勤務時間
9. 常勤換算値

**記入上の注意**:
- シフトタイプは事業所独自の略称でも可（例: 早、日、遅、夜、休）
- 勤務時間は「HH:mm ~ HH:mm」形式で記載
- 常勤換算値は小数点第2位まで記載

### 5.3 記入例

**シフト記号の例**:
- 早: 早番（8:00~16:00）
- 日: 日勤（9:00~18:00）
- 遅: 遅番（11:00~20:00）
- 夜: 夜勤（22:00~翌8:00）
- 休: 休日
- 明: 明け休み（夜勤後の休息）

**記入例**（11月の一部）:

| 氏名 | 職種 | 資格 | 1日 | 2日 | 3日 | 4日 | 5日 |
|------|------|------|-----|-----|-----|-----|-----|
| 田中太郎 | 介護職員 | 介護福祉士 | 早 | 日 | 遅 | 休 | 早 |
| 佐藤花子 | 看護職員 | 看護師 | 日 | 遅 | 休 | 早 | 日 |
| 鈴木一郎 | 介護職員 | 初任者研修 | 夜 | 明 | 休 | 日 | 遅 |

## 7. Phase 25実装時の注意点

### 7.1 データモデル設計

**予実データの保存**:
- `plannedShiftType`: 予定シフトタイプ
- `plannedStartTime`: 予定開始時刻
- `plannedEndTime`: 予定終了時刻
- `actualShiftType`: 実績シフトタイプ（任意）
- `actualStartTime`: 実績開始時刻（任意）
- `actualEndTime`: 実績終了時刻（任意）
- `breakMinutes`: 休憩時間（分）
- `notes`: 特記事項

### 7.2 コンプライアンスチェック実装

**人員配置基準チェック**:
- 事業所ごとの配置基準を設定UIで管理
- 日次でチェック
- 実績データがある場合は実績を使用、なければ予定を使用

**常勤換算計算**:
- 週所定労働時間を事業所設定で管理（デフォルト: 40時間）
- 実績勤務時間から常勤換算値を計算
- 月所定労働時間 = 週所定労働時間 × 4.33

**労基法チェック**:
- 休憩時間チェック: 実績データから勤務時間を計算
- 連続勤務制限チェック: 既存機能を活用
- 勤務間インターバルチェック: 既存機能を活用

### 7.3 Excel出力実装

**標準様式第1号（行政提出用）**:
- 予定シフトのみを出力
- フォーマットは厚生労働省の標準様式に完全準拠
- ファイル名: `勤務形態一覧表_YYYYMM.xlsx`

**内部管理用（予実2段書き）**:
- 予定と実績の両方を出力
- 差異をハイライト表示（オレンジ色）
- ファイル名: `勤務形態一覧表_予実_YYYYMM.xlsx`

## 8. よくある質問（FAQ）

### Q1: 常勤換算2.5人とは、具体的にどういう配置ですか？

**A**: 以下のような組み合わせが考えられます：

- パターン1: 常勤2人 + 非常勤1人（週20時間勤務）
- パターン2: 常勤3人（常勤換算3.0人）
- パターン3: 非常勤5人（各週20時間勤務、常勤換算2.5人）

### Q2: 夜勤後の「明け休み」は勤務日数にカウントされますか？

**A**: 「明け休み」は休日として扱われます。勤務日数にはカウントされません。

### Q3: 予定と実績が異なる場合、どちらを報告すべきですか？

**A**:
- **行政提出**: 予定シフトを記載（標準様式第1号）
- **運営指導（監査）**: 実績シフトを証明資料として提示

### Q4: 常勤換算値の計算で、有給休暇は勤務時間に含めますか？

**A**: はい、有給休暇は勤務時間としてカウントされます。

### Q5: 連続勤務制限は法律で定められていますか？

**A**: 労働基準法では明確な規定はありませんが、介護労働安全衛生ガイドラインで7日以内を推奨しています。事業所の就業規則で定めることが一般的です。
