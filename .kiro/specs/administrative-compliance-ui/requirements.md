# Requirements Document

## Project Description (Input)

Phase 61: 運営指導モード・月別書類管理 (administrative-compliance-ui)

介護事業所の行政対応（運営指導・監査）を強化するための機能。

## 背景
- 標準様式第1号のExcelエクスポートはPhase 25で実装済み
- 運営指導当日に「当月・過去月のシフト実績を素早く提示」「コンプライアンス状況を一覧表示」する専用UIが未実装
- 月別書類（勤務形態一覧表）を月ごとに保管・一括DLする機能が未実装

## 対象機能

### 1. 月別書類管理
- 過去の勤務形態一覧表（ExcelファイルへのFirestoreメタデータ記録）を月別に保管
- 月別書類一覧UI（年月別に生成済み書類を表示・再DL）
- 一括ダウンロード（複数月分をZIPで取得）

### 2. 運営指導モード
- 専用モードボタン（「運営指導モード」トグル）
- 当月の実績シフト一覧・コンプライアンスチェック結果をワンクリックで表示
- 過去12ヶ月分の実績一覧（月別概要）
- 常勤換算値・人員配置基準の充足状況をダッシュボード形式で表示

### 3. 電子申請フロー案内
- Excel出力後の提出手順をステップ形式で案内（GビズIDポータルへのアップロード手順）
- 案内はUI内モーダル（外部リンクなし）

## 技術制約
- Firestore に月別書類メタデータを保管（実ファイルはユーザーのローカル保存）
- 既存の complianceService.ts・exportExcel.ts を最大限活用
- 介護現場向け：シンプル・視認性重視・クリック数最小
