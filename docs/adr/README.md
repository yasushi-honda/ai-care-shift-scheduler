## ADRテンプレート

新規ADRは以下の形式で作成：

```markdown
# ADR-NNNN: タイトル

**日付**: YYYY-MM-DD
**ステータス**: 提案 | 採用 | 廃止 | 置換
**関連**: バグ番号、PR番号など

## コンテキスト
決定が必要になった背景

## 決定
採用した解決策

## 影響
- 正の影響
- 負の影響

## 参照
関連ドキュメントへのリンク
```
