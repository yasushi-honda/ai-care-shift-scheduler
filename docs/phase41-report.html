概要

Phase 41では、月次レポート機能の強化を実装しました。管理者向けの経営分析レポートと、スタッフ向けの個人勤務実績レポートを含む包括的なレポート機能を追加しました。

主な機能

ダッシュボード

月間サマリー、グラフ表示、PDF出力

勤務時間レポート

スタッフ別勤務時間、残業警告、日別詳細

シフト種別レポート

シフト種別分布、スタッフ別内訳

スタッフ稼働レポート

稼働統計、月間カレンダー

経営分析レポート

充足率、コスト推計、改善提案（管理者向け）

個人レポート

勤務実績、休暇残高（スタッフ向け）

技術的な特徴

Result型パターン

TypeScriptの型narrowingを正しく機能させるため、明示的な比較を使用：

if (result.success === true) {
  // result.data が利用可能
} else if (result.success === false) {
  // result.error が利用可能
}

レスポンシブ対応

ブレークポイント 表示形式
モバイル（767px以下） カード形式、横スクロールタブ
タブレット（768-1023px） 2カラムレイアウト
デスクトップ（1024px以上） フルテーブル表示

アクセス制御

ロール アクセス可能なタブ
管理者（facility-admin, manager） 全タブ + 経営分析
スタッフ 基本タブ + 個人レポート
