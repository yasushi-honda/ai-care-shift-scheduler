Phase 45: AIシフト生成進行状況表示機能

AI生成中の待ち時間に視覚的なフィードバックを提供

実装完了

機能概要

AIシフト生成処理は約2〜3分かかるため、ユーザーに処理状況を視覚的に伝える機能を実装しました。

主な機能

  • 5段階のステップ表示（進行状況の可視化）
  • プログレスバー（時間ベースの進捗表示）
  • 経過時間・残り時間の表示
  • キャンセル機能（確認ダイアログ付き）
  • 完了・エラー・キャンセル時のフィードバック

技術的アプローチ

クライアントサイド疑似プログレス方式

Cloud Functionの変更を必要とせず、経過時間に基づいて進行状況を推定表示します。

メリット

  • バックエンド変更不要
  • 既存のAPI設計を維持
  • シンプルな実装
  • アクセシビリティ対応

5段階の処理ステップ

  • 1

    リクエスト送信中 0秒〜

    サーバーにリクエストを送信しています...

  • 2

    AIがシフトを分析中 5秒〜

    スタッフ情報と制約条件を分析しています...

  • 3

    シフト案を生成中 30秒〜

    AIがシフト案を生成しています...

  • 4

    評価・最適化中 90秒〜

    生成されたシフトを評価・最適化しています...

  • 5

    結果を保存中 150秒〜

    シフトデータを保存しています...

AIGenerationProgress.tsx

メインコンポーネント。サブコンポーネントを統合し、状態に応じた表示を行います。

Props

プロパティ 型 説明
state GenerationProgressState 進行状況の状態
onCancel () => void キャンセル時のコールバック

useAIGenerationProgress.ts

プログレス状態を管理するカスタムフック。

返り値

プロパティ 型 説明
state GenerationProgressState 現在の状態
startGeneration () => void 生成開始
completeGeneration () => void 生成完了
failGeneration (msg) => void エラー終了
cancelGeneration () => void キャンセル
reset () => void リセット

ProgressSteps.tsx

5段階のステップを縦に並べて表示。現在のステップをハイライト、完了済みステップにチェックマークを表示。

機能

  • 現在のステップをパルスアニメーションで強調
  • 完了済みステップに緑のチェックマーク
  • ステップ間の接続線表示
  • レスポンシブ対応（モバイルで説明文を非表示）

ProgressTimer.tsx

経過時間と残り時間を表示。

表示パターン

  • 0〜50%: 「約2〜3分かかります」
  • 50〜100%: 「残り約 X:XX」
  • 100%超過: 「もう少しお待ちください...」

ProgressBar.tsx

プログレスバーを表示。経過時間に基づいて進捗率を計算。

機能

  • アニメーション付きのプログレスバー
  • シマーエフェクト（光沢の流れるアニメーション）
  • 進捗率のパーセンテージ表示
  • アクセシビリティ対応（ARIA属性）

型定義

type GenerationStatus = 'idle' | 'generating' | 'completed' | 'error' | 'cancelled';

interface StepDefinition {
  id: number;
  label: string;
  description: string;
  startTimeSeconds: number;
}

interface GenerationProgressState {
  status: GenerationStatus;
  currentStep: number;
  elapsedSeconds: number;
  estimatedTotalSeconds: number;
  errorMessage?: string;
}

使用方法

// App.tsxでの使用例
import { AIGenerationProgress } from './src/components/AIGenerationProgress';
import { useAIGenerationProgress } from './src/hooks/useAIGenerationProgress';

const App = () => {
  const aiProgress = useAIGenerationProgress();

  const handleGenerateClick = async () => {
    aiProgress.startGeneration();
    try {
      await generateShiftSchedule(...);
      aiProgress.completeGeneration();
    } catch (error) {
      aiProgress.failGeneration(error.message);
    }
  };

  return (
    <div>
      {aiProgress.state.status === 'generating' && (
        <AIGenerationProgress
          state={aiProgress.state}
          onCancel={() => aiProgress.cancelGeneration()}
        />
      )}
    </div>
  );
};
